# AI Learning Platform - 完整测试列表

## 项目概述

**AI教学展示平台** - 公司内部AI学习资料分享平台
- 技术栈: Next.js 14 + FastAPI + MySQL + MinIO
- 设计规范: Brutally Minimal (极致极简)
