## Affected Pages/Routes

- `/login` — 登录页面，JWT 认证流程
- `/` — 首页课件列表，分页和排序功能
- `/materials/:id` — 课件详情页，视频播放/PDF预览，浏览统计
- `/upload` — 文件上传页面，支持视频和PDF

## Key Interactions to Verify

### 认证流程
- 使用邮箱/密码登录成功，获取 JWT token
- Token 过期后自动刷新
- 未登录用户访问受保护页面重定向到登录

### 文件上传
- 选择 MP4/WebM 视频上传，显示进度
- 选择 PDF 文件上传
- 上传成功后跳转到课件详情页
- 上传过程中显示缩略图生成状态

### 课件浏览
- 首页列表按时间/浏览数/点赞数排序
- 分页功能（每页20条）
- 点击课件进入详情页
- 视频播放支持拖拽进度（Range请求）
- PDF 预览使用浏览器原生渲染

### 点赞功能
- 点击点赞按钮，点赞数+1
- 再次点击取消点赞，点赞数-1
- 点赞状态在页面刷新后保持

## Edge Cases

### 文件上传
- 上传超过 500MB 的视频，显示错误提示
- 上传不支持的格式（如 .exe），被拒绝
- 上传过程中网络中断， graceful 失败
- 同时上传多个大文件，检查并发控制

### 浏览统计
- 同一用户10分钟内重复访问，浏览数不增加
- 未登录用户按 IP 去重
- 高并发访问时浏览统计准确

### 缩略图生成
- 视频第一帧截取失败，显示默认占位图
- PDF 第一页提取失败，显示默认占位图
- 并发上传多个文件，缩略图队列正常工作

## Critical Paths

### 路径1: 完整上传流程
登录 → 进入上传页 → 选择视频 → 填写标题 → 上传 → 等待缩略图 → 查看详情

### 路径2: 浏览和互动
登录 → 首页浏览列表 → 查看视频 → 点赞 → 刷新页面验证状态保持

### 路径3: 定时清理
上传中断 → 数据库状态停留在 'processing' → 等待30分钟 → 定时任务清理记录和临时文件

## 测试优先级

### P0（阻塞发布）
- 登录认证
- 文件上传成功
- 视频/PDF 预览
- 浏览统计准确

### P1（重要）
- 点赞功能
- 分页排序
- 错误提示

### P2（优化）
- 缩略图生成
- 上传进度显示
