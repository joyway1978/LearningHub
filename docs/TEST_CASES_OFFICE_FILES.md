# Office 文件上传功能测试用例

> 本文档针对 PPT、DOCX、XLSX 文件上传、转换和显示功能的测试用例设计
> 创建时间: 2026-03-26

## 1. 功能概述

本功能支持用户上传 Office 文件（PPT、Word、Excel），后端使用 LibreOffice 将其转换为 PDF，然后通过 PDFViewer 在前端展示。

## 2. 测试环境

- **前端**: http://localhost:3000
- **后端**: http://localhost:8000
- **测试文件位置**: `/tmp/test_files/`

## 3. 测试用例详情

## 4. 测试数据准备

### 4.1 测试文件清单

| 文件名 | 类型 | 大小 | 描述 |
|--------|------|------|------|
| test_presentation.pptx | PPTX | 10MB | 标准演示文稿，20页 |

### 4.2 生成测试文件的脚本

```bash
# 创建测试目录
mkdir -p /tmp/test_files

# 创建空白 PPTX
echo -n "" > /tmp/test_files/empty.pptx

# 创建损坏的 PPTX
echo "corrupted data" > /tmp/test_files/corrupted.pptx

# 注：其他测试文件需要使用相应软件手动创建
```
