# 设计系统 — AI教学展示平台

## 产品上下文

- **产品名称**: AI教学展示平台
- **产品定位**: 公司内部AI学习资料分享平台
- **目标用户**: 公司内部员工（内容贡献者和学习者）
- **项目类型**: 内部工具/Web应用
- **技术栈**: Next.js 14 + Tailwind CSS + FastAPI

## 设计原则

1. **内容优先**: 界面不抢夺内容注意力，让员工专注于学习
2. **效率至上**: 信息架构清晰，支持快速发现和浏览
3. **反馈即时**: 点赞、浏览统计实时可见，给予上传者成就感
4. **克制装饰**: 用排版和色彩层级传达信息，而非装饰元素

## 美学方向

**Brutally Minimal（极致极简）**

- 白底黑字，高信息密度
- Typography 承担所有视觉工作
- 参考: Notion, Dropbox Paper, Linear

**装饰级别**: Minimal（极简）

- 无渐变、无阴影、无装饰性圆角
- 只有必要的分隔线和按钮

## 字体系统

### 字体家族
- **主字体**: Inter — 现代、清晰、屏幕优化
- **等宽字体**: JetBrains Mono — 代码片段、数据展示

### 字体层级

| 层级 | 大小 | 字重 | 行高 | 字间距 | 用途 |
|------|------|------|------|--------|------|
| H1 | 32px | 700 | 1.2 | -0.02em | 页面标题 |
| H2 | 24px | 600 | 1.3 | -0.01em | 区块标题 |
| H3 | 18px | 600 | 1.4 | 0 | 卡片标题、小节标题 |
| Body | 16px | 400 | 1.6 | 0 | 正文内容 |
| Small | 14px | 400 | 1.5 | 0 | 辅助文字、元信息 |
| Caption | 12px | 500 | 1.4 | 0.02em | 标签、小标题（大写） |

### 数字字体
- 使用 `font-variant-numeric: tabular-nums` 确保统计数字对齐
- 例: 浏览数、点赞数

## 动效系统

### 动画理念
**Minimal-Functional（极简功能型）** — 只有帮助理解的动画

### 时长
| 类型 | 时长 | 用途 |
|------|------|------|
| micro | 50-100ms | 按钮反馈、状态切换 |
| short | 150ms | 悬停效果、颜色过渡 |
| medium | 250ms | 页面切换、展开/收起 |

### 缓动函数
- **进入**: `ease-out` — 快速开始，缓慢结束
- **离开**: `ease-in` — 缓慢开始，快速结束
- **移动**: `ease-in-out` — 平滑加减速

### 具体应用
- 按钮悬停: 150ms ease-out
- 卡片悬停: 150ms ease-out，轻微上移 2px
- 页面切换: 250ms ease-out
- 列表加载: 渐进式淡入，每项延迟 50ms

## 页面设计

### 课件列表页

**布局**
- 顶部: 搜索栏 + 上传按钮
- 次级: 分类导航（全部/视频/PDF/我的上传）
- 主体: 卡片网格（3列 → 2列 → 1列响应式）

**卡片内容**
- 缩略图（视频播放图标/文档图标）
- 标题（最多2行）
- 统计行（浏览数 · 点赞数 · 上传者 · 时间）

### 课件详情页

**布局**
- 左侧 70%: 视频播放器/PDF预览
- 右侧 30%: 信息面板

**信息面板**
- 标题
- 描述
- 上传者信息
- 统计（浏览数、点赞数）
- 点赞按钮（琥珀色突出）
- 相关推荐

### 上传页

**布局**
- 中央: 拖拽上传区域
- 下方: 标题、描述输入
- 底部: 提交按钮

**上传状态**
- 进度条
- 缩略图生成状态

## 设计决策记录

| 日期 | 决策 | 理由 |
|------|------|------|
| 2026-03-23 | 采用 Brutally Minimal 美学 | 内部工具需要员工专注于内容而非界面 |
| 2026-03-23 | 选择 Inter 字体 | 屏幕优化、多语言支持好、现代感强 |
| 2026-03-23 | 琥珀色作为强调色 | 在冷静的专业感中加入温暖，点赞有满足感 |
| 2026-03-23 | 8px 基础间距 | Tailwind 默认单位，开发效率高 |
| 2026-03-23 | 极简动效 | 功能优先，动画只用于提升理解 |

## 开发实现

### Tailwind CSS 配置

```javascript
// tailwind.config.js 扩展
{
  theme: {
    extend: {
      colors: {
        primary: {
          DEFAULT: '#1a1a2e',
          light: '#2d2d44',
        },
        accent: {
          DEFAULT: '#f59e0b',
          hover: '#d97706',
        },
        surface: '#ffffff',
        background: '#fafaf9',
      },
      fontFamily: {
        sans: ['Inter', 'system-ui', 'sans-serif'],
        mono: ['JetBrains Mono', 'monospace'],
      },
      spacing: {
        '18': '4.5rem',  // 72px
        '22': '5.5rem',  // 88px
      },
    },
  },
}
```
